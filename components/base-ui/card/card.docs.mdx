### Overview

Basic card for displaying image (top, main section) plus content (lower section). 
Apply themeable css (see compositions) to style the card according to your design system.

Example:

```js live
    <div style={{maxWidth: 600}}>
        <Card 
            img={"https://wiki.mageia.org/mw-en/images/thumb/b/be/Mageia-7-Default-1920x1200.png/600px-Mageia-7-Default-1920x1200.png"}
        >
            Change the image url to adjust the background, or change the styles (from @dell/design.styles.dds-css) to adjust how it looks
        </Card>
    </div>


















```
