### Overview

This component is a concrete example of the [api context provider component](https://bit.dev/learn-harmony/movies/context/api-context-provider), with pre-populated
baseurl and api key.

These can both be overriden, for instance when wanting to test out a different data environment for the movies api (pre-production, laboratory, etc).

Example usage:

Provider side
``` js
app.js

import { MovieApiContextProvider } from '@learn-harmony/movies.movies.context.movies-api-context';

...
    <MovieApiContextProvider baseUrl={config.baseUrl} apiKey={config.apiKey}>
        <App />
    </ MovieApiContextProvider>

```

Consumer side:
```
import { MovieApiContextProvider } from '@learn-harmony/movies.movies.context.movies-api-context';



```
